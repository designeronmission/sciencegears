1. Introduction

Welcome to the Sciencegears Potentiostat user manual. This device is designed for electrochemical measurements including cyclic voltammetry (CV), electrochemical impedance spectroscopy (EIS), and more.

Quick Tip: Always ensure proper electrode connections before starting experiments.

2. Safety Information

  • Use only the provided power adapter.
  • Avoid exposing the device to moisture.
  • Ensure proper grounding.

  • Wear gloves when handling corrosive electrolytes.
  • Work in a well-ventilated area.

3. Technical Specifications

Parameter Value
Voltage Range ±10V
Current Range 1nA - 100mA
Input Impedance 1TΩ
Supported Techniques CV, LSV, EIS, Chronoamperometry
