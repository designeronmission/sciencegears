Squidstat Potentiostat Installation Guide

Complete setup and configuration for your multi-channel electrochemical workstation

Estimated time: 60-90 minutes
Tools required: USB cable, screwdriver, level
Safety rating: High voltage - caution required
Download PDF Watch Video Guide

Before You Begin

This guide provides complete installation instructions for your Squidstat Electrochemical Workstation. The Squidstat is a high-performance potentiostat/galvanostat with up to 16 independent channels for advanced electrochemical research.

Important Safety Notice

The Squidstat can deliver high voltages and currents. Always power off the unit when making connections. Ensure your work area is dry and properly grounded. Wear appropriate personal protective equipment when handling electrochemical cells.

What's Included

Your Squidstat package should contain the following components:

  • Squidstat main unit with power cord
  • USB 2.0 connection cable
  • EC-Lab Software license key and installation USB
  • Electrode cables (working, reference, counter)
  • Rack mounting kit (screws and brackets)
  • Quick start guide

System Requirements

Before installation, verify your computer meets these requirements:

  • Windows 10 or 11 (64-bit)
  • Intel Core i5 processor or equivalent
  • 8GB RAM minimum (16GB recommended)
  • USB 2.0 or 3.0 port
  • 500MB free disk space
  • Administrator privileges for software installation

Hardware Installation

1Unpacking and Inspection

Carefully remove all components from the packaging and inspect for any shipping damage. Verify all components against the packing list.

Choose a stable location for your potentiostat with adequate ventilation (minimum 10cm clearance on all sides). The surface should be level and vibration-free.

2Rack Mounting (Optional)

For rack installation, use the provided mounting brackets and screws. The Squidstat requires 3U of rack space (13.3cm height).

Ensure the rack is stable and properly grounded. Use all four mounting points for secure installation.

3Power Connection

Connect the provided power cord to the rear panel of the Squidstat. Plug into a properly grounded outlet (100-240V, 50/60Hz).

Do not power on the unit until all connections are complete.

Power Considerations

Use only the provided power cord. Ensure your power source provides clean, stable voltage. For optimal performance, consider using a line conditioner or UPS.

4Computer Connection

Connect the USB cable between the Squidstat and your computer. Use the USB 2.0 port on the rear panel for best performance.

For multi-unit setups, connect each Squidstat to a separate USB host controller.

5Electrode Connections

Connect the working, reference, and counter electrode cables to the appropriate front panel connectors.

Ensure all connections are secure before applying potential to your electrochemical cell.

Software Installation

1EC-Lab Installation

Insert the provided EC-Lab installation USB or download the latest version from the BioLogic website. Run the setup executable and follow the on-screen instructions.

When prompted, enter your license key. Select "VSP Series" when choosing hardware support.

2Driver Installation

After EC-Lab installation, connect the Squidstat and power it on. Windows should automatically detect and install the necessary drivers.

If automatic installation fails, manually install drivers from the EC-Lab installation directory (Drivers folder).

3Initial Configuration

Launch EC-Lab and navigate to Tools > Hardware Manager. The Squidstat should appear in the connected devices list.

Configure your measurement settings according to your experimental requirements. Perform a system test to verify all channels are functioning properly.

4Firmware Update (If Required)

Check for firmware updates in the Hardware Manager. If an update is available, follow the on-screen instructions to complete the process.

Do not interrupt power during firmware updates as this may damage the unit.

Initial Calibration & Testing

1Open Circuit Test

With no cell connected, measure the open circuit potential on all channels. Values should be within ±2mV of zero.

2Known Resistor Test

Connect a 1kΩ resistor between working and counter electrodes. Run a quick potentiodynamic scan (e.g., -10mV to +10mV at 1mV/s).

The measured current should follow Ohm's Law (I = V/R) with less than 1% deviation.

3Multi-channel Verification

If using multiple channels, verify independent operation by running different techniques on adjacent channels.

Troubleshooting

If any tests fail, check all connections and ensure proper grounding. Contact ScienceGears technical support if issues persist after basic troubleshooting.
